## About Claude

Claude is an advanced AI assistant developed by Anthropic with powerful language understanding and generation capabilities. Claude CN community provides Chinese developers with:

- Learning Paths: stage-based reading routes that move from basics to real systems
- Claude Code: AI-powered command-line development tool
- Source Analysis: Evidence-based source reading built from the real release package and source map
- Agent Skills: Reusable domain skill system
- Claude API: Complete API integration guide
- Quickstarts: Quick start project templates
## Core Features

### Development Tools

- Claude Code CLI: Complete development tasks through natural language
- Code Execution: Safe code execution environment
- Git Integration: Intelligent version control operations
### Skill System

- Pre-built Skills: PowerPoint, Excel, Word, PDF processing
- Custom Skills: Create domain-specific skills
- Skill Marketplace: Share and discover community skills
### Advanced Capabilities

- Extended Thinking: Deep reasoning ability
- Computer Use: Computer interface control
- Tool Use: Flexible tool invocation
- MCP Protocol: External system integration
## Technical Highlights

### 🚀 Extended Thinking

Enables Claude to perform deep reasoning on complex problems for more accurate solutions. Perfect for multi-step analysis scenarios.

### 💻 Computer Use

Allows Claude to directly interact with [ computer](#) interfaces for automated task execution. From screenshot recognition to click operations, fully automated.

### 🔧 Tool Use

Flexible tool invocation system that enables Claude to use various external tools and APIs, extending its capabilities.

### 🌐 MCP Integration

Connect external data sources and services through Model Context Protocol, giving Claude access to real-time information and professional systems.

## Learning Paths

If you are unsure which section should come first, start directly with [Learning Paths](https://claudecn.com/en/docs/learning-paths/).

### Beginner Path (1-2 Weeks)

- Install Claude Code and complete quick start
- Try 3-5 basic development tasks
- Understand Agent Skills concepts
- Browse Cookbook basic examples
### Intermediate Path (2-4 Weeks)

- Master Claude Code common workflows
- Create your first custom Skill
- Complete a Quickstart project
- Learn tool use and MCP integration
### Advanced Path (1-3 Months)

- Build complex multi-Skill workflows
- Integrate Claude API into existing systems
- Implement automation with Computer Use
- Participate in community contributions
