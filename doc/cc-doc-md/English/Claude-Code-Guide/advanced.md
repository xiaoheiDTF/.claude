# Advanced Guide

This section covers advanced features that empower you to extend Claude Code’s capabilities—including Skills, Subagents, Hooks, and MCP integration.

If you are still in the “get Claude Code working well for daily tasks” stage, check [Learning Paths](https://claudecn.com/en/docs/learning-paths/) first to confirm whether this section is the right next step.

## Capability Comparison

| Feature | Purpose | Scope |
| --- | --- | --- |
| **Skills** | Reusable prompts and guidelines | Auto-applied based on context |
| **Subagents** | Specialized agents for specific tasks | Isolated context |
| **Hooks** | Event-triggered script execution | Workflow automation |
| **MCP** | Connect external tools and data | APIs, databases, browsers |

## Learning Path

- Start with Agent Loop to understand the control flow behind tool use
- Start with MCP Servers to connect external tools
- Learn Skills to create reusable knowledge modules
- Master Subagents for complex multi-step tasks
- Use Hooks for workflow automation
## When this section becomes the right fit

- you already use Claude Code reliably for everyday work
- you are starting to care about reuse, automation, collaboration, and external integration
- you want to turn isolated usage into a stable engineering system
