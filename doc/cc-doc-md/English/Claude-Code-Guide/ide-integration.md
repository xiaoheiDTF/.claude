# IDE Integration

Claude Code integrates with popular IDEs, providing a graphical interface and convenient workflow enhancements.

## Supported IDEs
[VS Code](vscode/)
[JetBrains](jetbrains/)
[Chrome DevTools](chrome/)

## Selection Guide

| IDE | Features | Best For |
| --- | --- | --- |
| **VS Code** | Official extension, most complete | Daily development |
| **JetBrains** | IntelliJ IDEA, WebStorm, etc. | Java/Kotlin/Frontend |
| **Chrome** | DevTools integration | Frontend debugging |

## IDE vs CLI

- IDE extensions provide graphical UI, CLI provides command-line interface
- Both share conversation history and configuration
- Some advanced features (like MCP config) require CLI
- Run claude in IDE’s integrated terminal for CLI access
