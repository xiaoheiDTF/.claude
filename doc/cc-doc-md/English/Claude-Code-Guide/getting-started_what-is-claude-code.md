# What is Claude Code

Claude Code is an AI coding assistant that runs in your terminal. It understands your codebase, executes commands, edits files, and helps you complete development tasks through natural conversation.

## Core Capabilities

### Intelligent Code Understanding

Claude Code can read and understand your project structure, analyze code logic, and answer questions about your codebase.

### Natural Language Interaction

Describe what you want in plain language. Claude Code translates your intent into actual code changes.

### Safe File Operations

All file modifications require your explicit approval before execution, ensuring you maintain control.

### Git Integration

Seamless version control: create commits, manage branches, review changes, and resolve conflicts.

### Multi-Model Support

Access different Claude models (Sonnet, Opus, Haiku) based on task complexity and speed requirements.

## How It Works

- Start Claude Code in your terminal: claude
- Describe your task in natural language
- Claude analyzes your codebase and proposes solutions
- Review and approve changes
- Claude executes the approved operations
## When to Use Claude Code

- Exploring new codebases: “What does this project do?”
- Bug fixing: “There’s a login error, help me fix it”
- Feature development: “Add user authentication”
- Refactoring: “Refactor this module to use async/await”
- Documentation: “Generate API documentation”
- Code review: “Review my recent changes”
