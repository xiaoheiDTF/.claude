# First Conversation

This guide walks you through your first conversation with Claude Code.

## Starting a Session

Navigate to your project directory and start Claude Code:

```bash
cd /path/to/your/project
claude
```

## Understanding the Interface
You’ll see the welcome screen with:

- Session information
- Recent conversations
- Latest updates and tips
Type `/help` to see available commands.

## Your First Prompt

Start by understanding your codebase:

```
> What does this project do?
```

Claude will analyze your files and provide a summary.

## Common First Questions

```
> What technologies does this project use?
```

```
> Where is the main entry point?
```

```
> Explain the folder structure
```

## Making Your First Edit
Ask Claude to make a simple change:

```
> Add a hello world function to the main file
```

Claude will:

- Find the appropriate file
- Show proposed changes
- Ask for your approval
- Execute the edit
## Using Git

```
> What files have I changed?
```

```
> Commit my changes with a descriptive message
```

## Ending a Session
Type `exit` or press `Ctrl+C` to end the session.

## Tips

- Be specific: Instead of “fix the bug”, try “fix the login bug where users see a blank screen”
- Break down tasks: Handle complex tasks in steps
- Let Claude explore first: Ask Claude to analyze before making changes
