# 什么是 Claude Code

Claude Code 是 Anthropic 推出的智能编码助手，它不仅能生成代码，更能理解整个项目、自主规划任务、调用工具完成复杂的开发工作。

## 从代码补全到智能体

[ 编程](#)辅助工具经历了三个发展阶段：

| 阶段 | 特点 | 代表产品 |
| --- | --- | --- |
| **代码补全** | 根据上下文预测下一行代码 | GitHub Copilot 早期版本 |
| **对话助手** | 通过对话回答编程问题、生成代码片段 | ChatGPT、Claude.ai |
| **智能体系统** | 理解项目全貌、自主规划、执行多步任务 | Claude Code |

Claude Code 属于第三阶段的智能体系统。它能够：

- 阅读和理解整个代码库
- 将复杂任务分解为可执行的步骤
- 调用终端命令、编辑文件、运行测试
- 根据执行结果调整后续行动
## 核心能力

### 项目级理解

Claude Code 不只看你当前编辑的文件，它会分析整个项目的结构、依赖关系和代码风格。当你说"重构认证模块"时，它知道去哪里找相关代码，也知道修改会影响哪些地方。

### 自主规划

面对复杂任务，Claude Code 会先制定计划，再逐步执行。你可以看到它的思考过程，也可以在任何时候介入调整方向。

### 工具调用

Claude Code 能够：

- 读写文件
- 执行终端命令
- 运行测试
- 搜索代码库
- 使用 Git 进行版本控制
### 环境感知

Claude Code 了解你的开发环境——使用什么语言、什么框架、什么工具链。它生成的代码会符合项目的既有风格和规范。

### 记忆能力

通过 CLAUDE.md 等机制，Claude Code 可以记住项目的特定约定和你的偏好设置，在后续对话中保持一致性。

## Claude Code 与 Claude 模型

这是两个不同的概念：

- Claude 模型：Anthropic 开发的大语言模型（如 Claude Sonnet、Claude Opus），是底层的 AI 能力
- Claude Code：基于 Claude 模型构建的开发工具，添加了文件操作、命令执行、项目理解等能力
可以这样理解：Claude 模型是"大脑"，Claude Code 是"大脑 + 手脚 + 工具箱"。

## 适合谁使用

Claude Code 适合：

- 专业开发者：提高开发效率，减少重复劳动，专注于更有价值的工作
- 学习 编程的人：获得即时的代码解释和指导，加速学习过程
- 技术人员：即使不是专职开发，也能完成脚本编写、数据处理等技术任务
## 能做什么

一些典型的使用场景：

- 快速理解陌生代码库
- 编写新功能
- 调试和修复 Bug
- 重构代码
- 编写测试
- 生成文档
- 自动化重复任务
## 局限性

需要了解的限制：

- 对于超大型代码库，理解深度可能受限
- 复杂的架构决策仍需人类判断
- 生成的代码需要审查和测试
- 无法替代对编程原理的学习和理解
