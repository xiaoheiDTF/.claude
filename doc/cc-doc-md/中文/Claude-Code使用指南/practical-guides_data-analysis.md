# 数据分析

Claude Code 能够处理表格数据、进行[ 统计](#)分析、生成可视化报告。

## 表格数据处理

### 数据清洗

```
@销售数据.csv 清洗这份数据：删除重复行、填充空白单元格、统一日期格式为YYYY-MM-DD
```

### 数据转换

```
@原始数据.xlsx 把金额列转换成数字格式，保留2位小数
```

### 数据合并

```
@数据1.csv @数据2.csv 合并这两份数据，按客户ID去重
```

## 统计分析

### 基础统计

```
@销售数据.csv 计算：总销售额、平均订单金额、最高/最低销售额
```

### 分组统计

```
@订单数据.csv 按月份统计销售额，并计算环比增长率
```

### 趋势分析

```
@历史数据.csv 分析过去12个月的销售趋势，找出增长最快的产品类别
```

## 数据提取

### 从文档提取表格

```
@报告.pdf 提取文件中的所有表格数据
```

### 提取特定字段

```
@客户列表.xlsx 提取所有客户的姓名、电话、邮箱，生成联系人清单
```

### 条件筛选

```
@销售记录.csv 筛选出金额大于10000的订单
```

## 报告生成

### 数据摘要

```
@季度数据.xlsx 生成一份数据摘要，包含关键指标和主要发现
```

### 分析报告

```
@销售数据.csv 生成销售分析报告，包含：
1. 整体业绩概览
2. 产品销售排名
3. 区域分布分析
4. 趋势预测和建议
```

### 对比分析

```
@2023数据.csv @2024数据.csv 对比两年的数据，分析变化趋势
```

## 实战案例

### 案例：销售数据分析
场景：分析月度销售数据，找出最佳表现。

```
@sales_data.csv 分析这份销售数据，找出：
1. 销售额最高的月份和产品
2. 增长最快的产品类别
3. 需要关注的下滑趋势
```

分析结果示例：

**整体表现**

- 总销售额：1,234,567元
- 月均销售额：102,880元
- 最高月份：6月（150,000元）
**产品分析**

- 最畅销产品：产品A（占比40.5%）
- 增长最快：产品C（增长100%）
- 需关注：产品B（连续3个月下滑）
### 案例：客户数据整理

场景：整理客户流失数据，生成分析报告。

```
@客户数据.xlsx 分析客户流失情况：
1. 流失率趋势
2. 流失原因分布
3. 高风险客户特征
4. 改进建议
```

### 案例：财务数据汇总
场景：汇总多个部门的费用数据。

```
@部门A费用.xlsx @部门B费用.xlsx @部门C费用.xlsx 
汇总各部门费用，按类别统计，生成费用对比表
```

## Excel 专项处理

### 公式检查

```
@财务表.xlsx 检查所有公式是否正确，找出可能的计算错误
```

### 数据透视

```
@订单明细.xlsx 按产品类别和月份生成数据透视表
```

### 格式美化

```
@报表.xlsx 美化表格格式：添加表头样式、设置条件格式、调整列宽
```

## 效率技巧

- 明确分析目标：说清楚想要什么结论
- 指定输出格式：表格、列表、图表描述
- 分步处理：复杂分析可以拆分成多个步骤
- 验证结果：重要数据建议人工复核
- 保存模板：常用的分析流程可以模板化
