## 预计时间

完成整个入门系列大约需要 30-60 分钟。你也可以根据自己的需求跳过已熟悉的章节。

## 准备工作

在开始之前，请确保：

- 拥有 Anthropic 账户 或 Claude Pro/Max 订阅
- 系统满足基本要求（详见安装配置章节）
- 有基础的命令行使用经验（可选，但有帮助）
