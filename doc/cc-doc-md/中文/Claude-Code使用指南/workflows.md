## 工作流概览

### 上下文管理
Claude Code 的性能与你提供的上下文质量直接相关。过多无关信息会降低响应准确性，而精准的上下文能让 Claude 快速理解任务。

### 代码修改

从规划模式到精确指令，掌握让 Claude 按你的意图修改代码的技巧。学习如何使用截图、中断和历史编辑功能。

### Git 集成

Claude Code 深度集成 Git，支持智能提交信息生成、Pull Request 创建、代码审查处理等完整的版本控制工作流。

### 高效实践

处理复杂任务的策略，包括检查清单、草稿本、批量处理等高级技巧，以及常见场景的最佳实践。

## 这部分最适合什么阶段

- 你已经完成基础安装和第一次真实任务
- 你开始重复做相似类型的开发工作
- 你希望把“偶尔有效”变成“稳定高效”的工作方式
