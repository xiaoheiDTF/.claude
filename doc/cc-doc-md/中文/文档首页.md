## 关于 Claude
Claude 是 Anthropic 开发的先进 AI 助手，具备强大的语言理解和生成能力。这个文档区重点服务几类常见需求：想找 `Claude 中文文档` 的读者、想系统学习 `Claude Code` 的开发者、想看 `Claude Skills 示例` 的实践用户，以及想快速验证产品思路的团队。Claude 中文社区致力于为中文开发者提供：

- 学习路径: 按阶段和目标组织的由浅入深阅读路线
- Claude Code: AI 驱动的命令行开发工具
- 源码分析: 基于真实发布包和 source map 的结构化源码研究入口
- Agent Skills: 可复用的领域技能系统
- Claude API: 完整的 API 集成指南
- Quickstarts: 快速上手项目模板
## 核心特性

### 开发工具

- Claude Code CLI: 通过自然语言对话完成开发任务
- Code Execution: 安全的代码执行环境
- Git 集成: 智能的版本控制操作
### 技能系统

- 预构建技能: PowerPoint、Excel、Word、PDF 处理
- 自定义技能: 创建专属领域技能
- 技能市场: 共享和发现社区技能
### 高级能力

- Extended Thinking: 深度推理能力
- Computer Use:  计算机界面控制
- Tool Use: 灵活的工具调用
- MCP Protocol: 外部系统集成
## 技术亮点

### 🚀 扩展思维（Extended Thinking）

让 Claude 在复杂问题上进行深度推理，提供更准确的解决方案。适用于需要多步骤分析的场景。

### 💻 计算机控制（Computer Use）

允许 Claude 直接操作计算机界面，实现自动化任务执行。从截图识别到点击操作，全程自动化。

### 🔧 工具调用（Tool Use）

灵活的工具调用系统，让 Claude 能够使用各种外部工具和 API，扩展其能力边界。

### 🌐 MCP 集成

通过 Model Context Protocol 连接外部数据源和服务，让 Claude 访问实时信息和专业系统。

## 学习路径推荐

如果你不确定该先看哪一部分，直接从 [学习路径](https://claudecn.com/docs/learning-paths/) 开始会更高效。

### 入门路径（1-2 周）

- 安装 Claude Code 并完成快速开始
- 尝试 3-5 个基本开发任务
- 了解 Agent Skills 概念
- 浏览 Cookbook 基础示例
### 进阶路径（2-4 周）

- 掌握 Claude Code 常用工作流
- 创建第一个自定义 Skill
- 完成一个 Quickstart 项目
- 学习工具调用和 MCP 集成
### 高级路径（1-3 月）

- 构建复杂的多 Skill 工作流
- 集成 Claude API 到现有系统
- 使用  Computer Use 实现自动化
- 参与社区贡献和最佳实践分享
