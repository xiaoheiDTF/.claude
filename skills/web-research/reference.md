## 一、来源质量评估框架

### CRAAP 评估标准

| 维度 | A（优秀） | B（良好） | C（一般） | D（待验证） |
|------|----------|----------|----------|------------|
| **可信度** | 官方文档、权威期刊、核心维护者 | 知名技术博客（Medium Top）、GitHub 高星项目 | 普通博客、StackOverflow 回答 | 个人博客、匿名论坛、无署名 |
| **时效性** | 最近 6 个月内 | 最近 1 年内 | 最近 2 年内 | 2 年前或不详 |
| **准确性** | 有代码示例、官方数据支撑 | 有引用来源、逻辑清晰 | 有观点但缺数据支撑 | 无依据、主观判断 |
| **相关性** | 直接相关，内容全面 | 相关，覆盖主要方面 | 部分相关 | 仅间接相关 |

### 来源优先级排序

```
1. 官方文档 / RFC / 规范        → A
2. 核心维护者/作者的技术文章     → A
3. 知名公司技术博客             → B
4. 高赞社区回答（100+ 赞）      → B
5. 技术教程网站                 → B
6. 普通开发者博客               → C
7. 论坛/社交媒体讨论            → C
8. 匿名/无署名内容              → D
```

### 来源评估示例

```markdown
| 来源 | 质量等级 | 时效性 | 理由 |
|------|---------|--------|------|
| [React 官方文档](https://react.dev) | A | 2024 | 官方权威来源 |
| [State of JS 2024](https://stateofjs.com) | A | 2024 | 年度权威调研 |
| [Dan Abramov Blog](https://overreacted.io) | A | 2023-2024 | React 核心团队成员 |
| [某 Medium 文章](https://medium.com/...) | B | 2024 | 高赞技术文章 |
| [某知乎回答](https://zhihu.com/...) | C | 2023 | 普通社区回答 |
```

## 六、搜索策略示例

### 示例 1：技术调研 — Playwright 浏览器自动化

```
搜索轮次：
1. "Playwright official documentation" → 获取官方权威信息
2. "Playwright best practices 2025" → 最新最佳实践
3. "Playwright vs Selenium comparison" → 了解竞品对比
4. "Playwright common issues pitfalls" → 常见问题
5. "Playwright 使用教程 实践" → 中文社区经验

深度阅读选择：
- 官方文档首页 + 核心概念页（A 级）
- Microsoft（维护者）的介绍博客（A 级）
- 高赞 Medium 实践文章（B 级）
- 知乎/掘金的中文实战经验（C 级）
```

### 示例 2：方案对比 — Electron vs Tauri

```
搜索轮次：
1. "Electron vs Tauri comparison 2025" → 直接对比
2. "Electron vs Tauri performance benchmark" → 性能数据
3. "Electron vs Tauri 选型" → 中文社区选型讨论
4. "Tauri production experience issues" → Tauri 生产使用问题
5. "Electron production issues 2024" → Electron 生产使用问题

深度阅读选择：
- Tauri 官方文档 + FAQ（A 级）
- Electron 官方文档（A 级）
- GitHub Issues 中的性能对比（B 级）
- 知乎选型讨论帖（C 级）
```

### 示例 3：通用资料收集 — AI Agent 架构设计

```
搜索轮次：
1. "AI agent architecture design patterns" → 核心概念
2. "LLM agent framework comparison" → 框架对比
3. "AI agent 架构设计 模式" → 中文资源
4. "autonomous agent design principles" → 设计原则
5. "AI agent 最新进展 2025" → 最新动态

深度阅读选择：
- 学术论文/综述（A 级）
- LangChain/AutoGen 官方文档（A 级）
- 知名研究者的博客（B 级）
```

## 七、搜索工具参数

### mcp__web-search-prime__web_search_prime

```
参数：
- search_query: 搜索关键词（英文搜索效果更好）
- content_size: "high"（获取详细摘要）
- location: "cn"（中文搜索）或 "us"（英文搜索）
- search_recency_filter: "oneYear" | "oneMonth" | "oneWeek" | "noLimit"
```

### mcp__web-reader__webReader

```
参数：
- url: 目标网页 URL
- return_format: "markdown"（推荐）
- retain_images: false（研究报告不需要图片）
- with_links_summary: true（获取页面中的链接，可能发现更多来源）
```
